--- layout: problem category: problems difficulty: 30 title: "Polinomial Dari Bilangan Fibonacci" problemId: 435 answer: -1 ---

Bilangan Fibonacci {fn, n ≥ 0} didefinisikan secara rekursif sebagai fn = fn-1 + fn-2 dengan dasar f0 = 0 dan f1 = 1.

Dinyatakan polinomial {Fn, n ≥ 0} sebagai Fn(x) = ∑fixi untuk 0 ≤ i ≤ n.

Sebagai contoh, F7(x) = x + x2 + 2x3 + 3x4 + 5x5 + 8x6 + 13x7, dan F7(11) = 268357683.

Misalkan n = 1015. Carilah hasil penjumlahan [∑0≤x≤100 Fn(x)] mod 1307674368000 (= 15!).
