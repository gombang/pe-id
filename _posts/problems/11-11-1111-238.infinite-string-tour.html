Buatlah sebuah barisan bilangan dengan alat pembuat bilangan acak "Blum Blum Shub":

s0 = 14025256
sn+1 = sn2 mod 20300713

Lalu gabungkan bilangan-bilangan  s0s1s2… untuk membuat sebuah string w dengan panjang tak hingga.
Maka, w = 14025256741014958470038053646…

Untuk bilangan bulat positif k, jika tidak terdapat substring dari w yang hasil penjumlahan angka-angkanya sama dengan k, p(k) akan dinyatakan sebesar nol. Jika setidaknya terdapat satu substring dari w yang memiliki hasil penjumlahan angka-angkanya sama dengan k, kita nyatakan p(k) = z, dimana z adalah posisi mulai dari substring tersebut.

Sebagai contoh:

Substrings 1, 14, 1402, …
dengan hasil penjumlahan angka masing-masing sama dengan 1, 5, 7, …
dimulai dari posisi 1, sehingga p(1) = p(5) = p(7) = … = 1.

Substrings 4, 402, 4025, …
dengan hasil penjumlahan angka masing-masing sama dengan 4, 6, 11, …
dimulai dari posisi 2, sehingga p(4) = p(6) = p(11) = … = 2.

Substrings 02, 0252, …
dengan hasil penjumlahan angka masing-masing sama dengan 2, 9, …
dimulai dari posisi 3, sehingga p(2) = p(9) = … = 3.

Perhatikan bahwa substring 025 yang dimulai dari posisi 3, memiliki hasil penjumlahan angka sama dengan 7, namun sudah terdapat substring lain (dimulai dari posisi 1) yang telah memiliki hasil penjumlahan angka 7, sehingga p(7) = 1, bukan 3.

Kita dapat membuktikan bahwa, untuk 0 < k ≤ 103, ∑ p(k) = 4742.

Carilah ∑ p(k), untuk 0 < k ≤ 2·1015.
