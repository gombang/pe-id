Perhatikan bilangan 3600. Bilangan ini sangat istimewa, karena

3600 = 482 +     362

3600 = 202 + 2×402

3600 = 302 + 3×302

3600 = 452 + 7×152

Dengan cara yang sama, kita dapat menemukann bahwa 88201 = 992 + 2802 = 2872 + 2×542 = 2832 + 3×522 = 1972 + 7×842.

Pada tahun 1747, Euler membuktikan bahwa terdapat bilangan yang dapat direpresentasikan sebagai penjumlahan dua bilangan kuadrat. Kita akan memperhatikan bilangan n yang bisa memenuhi keempat tipe representasi berikut ini:

n = a12 +   b12

n = a22 + 2 b22

n = a32 + 3 b32

n = a72 + 7 b72,

di mana ak dan bk adalah bilangan bulat positif.

Terdapat 75373 buah bilangan seperti di atas yang tidak melebihi 107.
Berapa banyak bilangan seperti di atas yang kurang dari 2×109?
