Misal G(a, b) adalah bilangan bulat non negatif n paling kecil, dimana fpb(n3 + b, (n + a)3 + b) adalah sebesar mungkin.
Sebagai contoh, G(1, 1) = 5 karena fpb(n3 + 1, (n + 1)3 + 1) mencapai nilai maksimal 7 untuk n = 5, dan akan mendapat nilai lebih kecil untuk 0 ≤ n < 5.
Misal H(m, n) = Σ G(a, b) untuk 1 ≤ a ≤ m, 1 ≤ b ≤ n.
Anda diberikan informasi H(5, 5) = 128878 dan H(10, 10) = 32936544.

Carilah H(18, 1900).
